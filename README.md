# Proxy

一个iOS代理应用，提供类似Shadowrocket的功能。

## 功能特性

- VPN配置管理
- 代理规则配置
- 服务器配置管理
- 流量转发
- 网络状态监控

## 系统要求

- iOS 14.0+
- Xcode 13.0+
- Swift 5.0+

## 安装说明

1. 克隆项目
2. 使用Xcode打开`Proxy.xcodeproj`
3. 配置开发者证书
4. 编译运行

## 注意事项

本应用需要以下权限：
- Network Extension
- VPN Configuration
- Background Mode
